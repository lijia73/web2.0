Course Project -- My Achievements

Skip to end of metadata
  • Page restrictions apply
  • Attachments:5
  • Added by 王青, last edited by 王青 on Jan 23, 2016
Go to start of metadata

本作业为寒假作业，不计入课程分数，但影响课程排名，提交时间为 2016年2月23日，提交时间和方式：和其它作业一样，使用M.A.提交，进行评审和评分。

注意：本作业分数将占整个排名分数的30%，其余作业一共占70%

题目内容

综合运用本课程学习的知识，实现一个和我们在课程中用到的M.A.系统一样的，My Achievements系统。

学生可以通过此系统：

  1. 提交作业
  2. 评审作业
  3. 查看自己以往的作业、分数和排名

老师和TA可以通过此系统：

  1. 布置作业
  2. 评审作业

作业流程

系统概念

功能与界面

学生用户的功能与界面，自行参照现有M.A.的实现。

TA与教师用户的功能与界面：1）布置作业；2）查看评审作业，请大家自行设计。

要求

本题目为开放性题目，不要求完整实现所有功能。实现多少，得对应分数。

  1. 在提交的项目文件中，附上README文件
    1. 说明本项目实现的功能点，以及项目的亮点，以利Reviewer评分
    2. 给出项目的安装运行方法
    3. 给出项目预设的管理员账号、密码
  2. 上述流程不能够更改
  3. 系统概念帮助大家理解需求，具体数据实现可以有不同

评分标准

功能要求（满分115分）

  1. （10分）发布作业（要求）
  2. （10分）查看作业（要求）、提交作业
  3. （20分）学生互评
  4. （10分）TA评审
  5. （10分）学生修改提交
  6. （10分）教师查看作业情况
  7. （10分）教师判定分数
  8. （10分）学生查看结果，包括查看自己所有以往作业情况

每个功能点评审时，reviewer应该判定为：1）未实现（0%）；2）完全实现（100%）；3）实现但有缺陷（xx%，xx根据缺陷，reviewer自行量裁）

额外加分（10~25分）

  1. 8个功能点完全实现5，或完全实现3；有缺陷3（最小xx% > 50%），获额外10分加分
  2. 在1的基础上，每增加一个完全实现，额外加5分

质量要求（扣分项，最多扣50分）

  1. CSS代码符合课程要求（一处不合格，扣1分，最多扣5分）
  2. HTML（Jade）代码符合课程要求（一处不合格，扣1分，最多扣5分）
  3. JavaScript代码符合课程要求（一处不合格，扣1分，最多扣10分）
  4. 数据持久化（数据未持久化扣30分）

    1. Feb 17, 2016

      王青

      Angular是浏览器端框架。当你在浏览器中切换url时，Angualr通过HTML history API侦测到，并执行相应JS，页面能够正确渲染。而当你refresh时，浏览器向server发送请求，server并不知道你的URL并不是真正的server page。

      仔细阅读：https://docs.angularjs.org/guide/$location，这种情况，问题在server，需要设置Rewrite规则（apache），或者catch all request handler （express等），向浏览器返回index.html

      Server side

      Using this mode requires URL rewriting on server side, basically you have to rewrite all your links to entry point of your application (e.g. index.html). Requiring a <base> tag is also important for this case, as it allows Angular to differentiate between the part of the url that is the application base and the path that should be handled by the application.

       

      这个问答和你情况一样，http://stackoverflow.com/questions/16569841/angularjs-html5-mode-reloading-the-page-gives-wrong-get-request

      这里也很有帮助：https://github.com/angular-ui/ui-router/wiki/Frequently-Asked-Questions#how-to-configure-your-server-to-work-with-html5mode
